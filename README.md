# Поле чудес

Всеми любимая телевизионная игра возвращается в своем новом обличии!

## Сборка
```
mkdir build
cd build
cmake ..
make
```

## Использование
```
./game_server <или> ./game_client
```

Сервер поддерживет работу сразу с несколькими клиентами.
